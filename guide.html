使用說明

點每個標題可以展開。這個 App 完全離線、資料只存在你的裝置。

🔀 模式切換(英文日常／多益／日文)

畫面右上角的「模式」可以切換三種學習內容,彼此進度分開:

  • 英文(日常):生活常用單字與會話。
  • 多益:多益情境詞庫、證照分級(綠/藍/金)。
  • 日文:N5 單字、五十音、N5 文法。

切換後,首頁、複習、測驗、文法都會跟著換成該模式的內容。

🃏 複習(背單字的核心)

用「間隔重複(FSRS)」在最該複習的時機把單字排出來。看完單字翻到背面,按四個鍵之一:

  • 複習與新字分兩段:一輪一定是「先清到期的複習 → 再學新字」,不會複習到一半突然冒出生字。進度列隨時顯示你在哪一段(例如「複習 3/20・新字 5 個在後面」)。
  • ✅ 分隔卡:到期複習做完時會先停下來問你「接下來有 N 個沒背過的新字,現在要學嗎?」——可以選開始學新字,或今天到此為止。
  • ⏸ 今天不學新字:累了或時間少就勾起來,今天只清複習(首頁待複習數也會跟著只算複習)。明天自動恢復,不會不小心永遠關著。
  • 🆕 第一次見到 / 🌱 學習中:卡片上會標示,讓你知道這張不是熟字——答不出來很正常,不用有壓力。
  • 重來=不記得、困難=有點難、良好=記得、簡單=很簡單。App 依此決定下次何時再出現。
  • ↩ 上一張:按錯評分可以連續退回好幾張。
  • 🔄 翻回正面、跳過這張(已經會的字不再出現,可在牌組頁恢復)。
  • 🔥 弱點加強:集中複習「常忘記、而且現在還沒記牢」的字(複習按過「重來」＋測驗答錯合計 2 次以上)。畢業條件:按「良好/簡單」把它的下次複習間隔養到 21 天以上,就會自動從清單消失——不用手動處理。(注意:忘記的次數是歷史紀錄,不會因為後來記住就減少,所以是看「現在記牢了沒」來決定畢不畢業。)
  • ⭐ 重點字:把想加強的字加星號,並可「只複習重點字」。
  • 🔁 反向複習:先看中文意思、回想英文(練「用得出來」)。
  • 自動發音、例句下的生字小詞典、搭配詞組🔊朗讀。
  • 🧠 背單字技巧:同樣複習一輪、怎麼背差很多的訣竅——①用詞組背不要背單字(卡片的「🔗 常見搭配」欄位就是給你這個用的)、②同字根一整家一起背(「詞性變化」欄位)、③意思很近的字不要同一天硬吃、④能拆的字先拆字根、⑤真的記不起來才用諧音關鍵字法。
  • 🧠 我的記憶法:翻到背面可以幫這個字寫自己的聯想(例如「ambulance = 唉唷不能死」),之後每次複習到它都會看到。
  • 📖 複習過的單字:可展開清單,分「今天/全部」回顧背過哪些字(附下次複習時間),點單字看完整詳情與複習歷史。

上方還能設定「每日新字上限」與「目標證照」(多益模式)。

📈 每天背幾個新字最好?

先懂一個關鍵:今天背的每一個新字,之後幾週會回來找你複習大約 7~10 次(FSRS 自動排程)。 所以「每日新字上限」真正決定的不是今天的量,而是之後每天要扛的複習量:

  • 10 個/天 → 穩定後每天約複習 70~100 張(約 15~20 分鐘)——時間零碎也撐得住。
  • 20 個/天(預設) → 約 140~200 張(約 30~40 分鐘)——多數人的甜蜜點。
  • 40 個/天 → 約 280~400 張(1 小時以上)——考前衝刺、時間充裕才建議。
  • 100 個以上/天 → 複習量會滾成每天數小時,幾乎沒有人長期撐得住。大部分人放棄背單字,就是敗在兩三週後的複習雪崩,而不是「記性不好」。

怎麼找到「你自己的」最佳量?每個人記憶力不同,與其套別人的數字,不如用這兩條規則自動調整(比任何建議都準):

  • 新字常按「重來」(超過三分之一)、或到期複習連續 3 天清不完 → 新字量砍半。
  • 幾乎都按「良好/簡單」、每天都清得完 → 可以加兩成試試。

短期衝刺想一天吃 200 個?可以,但要照規矩來,而且只適合有明確截止日(例如兩週後考試):

  • 一段上限 20~25 個新字、約 25~30 分鐘——一口氣塞 50 個,後面的會把前面的擠掉(工作記憶的極限)。
  • 200 個 = 8~10 段,分散在早、中、晚,段與段之間隔 1~2 小時讓記憶沉澱。
  • 睡前留一段:睡眠是免費的記憶固化器,睡前背的隔天記得最牢。
  • 每段先清到期複習、再學新字;衝刺結束後把上限調回 20~30,讓系統慢慢消化。
  • 要有心理準備:衝刺期的記憶保留率一定比細水長流低,考完要靠複習把它撿回來。

👉 在複習頁調「每日新字上限」時,下方會即時估算穩定後每天的複習量與時間——調之前先看一眼再決定。

🎯 不同詞性,用不同力氣背(多益提分關鍵)

很多人備考的錯誤是「每個字都用同樣力氣背」——結果簡單的名詞浪費時間,關鍵的動詞搭配又記不熟。 正確做法是看到單字先花 3 秒判斷它是哪類(卡片背面也會顯示對應提示):

  • 單純的名詞(invoice 發票、warehouse 倉庫):意思一對一、沒陷阱,記中文就好,快速過。多益考名詞大多只考「認不認識」。
  • 動詞:永遠不要單獨背,至少帶一個受詞——背 waive 就記 waive a fee(免收費用)、背 conduct 就記 conduct a survey(進行調查)。多益 Part 5 的動詞題幾乎都是搭配題:四個選項中文意思都通,只有一個是母語者習慣的搭配。卡片的「🔗 常見搭配」欄位就是為這個準備的。
  • 詞組(in advance、be subject to、market share):整組當一個大單字背,拆開背反而低效,考題裡它們永遠整組出現。

順帶解惑:為什麼 service fee 標 phr.、market share 卻標 n.?——這只是編字典的人標記習慣不同,兩種在文法上都是「複合名詞」,用法完全一樣,多益不會考這個差別。看到兩個字的組合,一律整組背意思和用法就對了,標記是 phr. 還是 n. 完全不用糾結。

另外,英英定義對多益幫助有限(多益不考定義,考的是在商業句子裡瞬間反應出意思), 所以卡片背面把例句和常見搭配放在前面、英英定義放到後面——複習時先看例句和搭配就好。

👉 想實戰演練搭配題:到「文法」頁的搭配專項①〜④(費用與權利、增減升降、會議與文件、綜合驗收)。

📝 測驗

由你的單字自動出題,四種模式:看字、🎧聽力、✍️填空(例句挖空)、⌨️聽寫(聽發音打單字)。可依牌組/情境/難度篩選,也可用「📅 複習日期」只考某一天(今天/昨天/更早)複習過的字。

  • 🧠 記憶階段篩選:可以只考「🆕 還沒記牢」的字(FSRS 排程間隔未滿 7 天),不把時間花在已經熟的字上;也可以反過來只考「🎓 已記牢」的字驗收。
  • 🆕 初見日期篩選:只考某一天「第一次見到」的字(第一次評分的那天),可選今天/昨天/更早——已經學過幾天的字絕不混進來。剛背完當天馬上主動回想一次,是記憶效果最好的時機。複習頁「📖 複習過的單字」裡也有「🆕 第一次見到」頁籤可以回顧每天初見了哪些字(純紀錄,不影響複習排程)。
  • 一輪制:範圍內每個字只考一次,考完看成績,不會重複出題。
  • 🔥 錯題重練:答錯的字進錯題清單(四種模式各自一份、跨日保留),之後答對就移出。
  • 🔤 加考詞性(看字/聽力):順便選這個字的詞性,可複選;多詞性的字要選齊才算對。詞性沒答對的字也會進錯題重練(就算意思選對了)。
  • 選項顯示英文(看字/聽力):答完後每個中文意思選項旁會標出對應的英文字——選錯時就看得到「你選的那個意思其實是哪個字」,連干擾選項也順便學。
📰 閱讀(用學過的單字讀短文)

每篇短文都只用你單字庫裡的字＋最基礎的常用字寫成,讓你「讀得懂、又剛好練到學過的字」:

  • 解鎖制:一篇文章的目標單字你學到 8 成就解鎖——複習越多,能讀的文章越多。
  • 點字查意:內文學過的字會上色,點一下看意思和例句;😵 忘了?按一下就把那個字排回今天的複習(10 分鐘內再出現)。
  • ✍️ 挖空測驗:同一篇文章把目標單字挖掉變選擇題,讀＋考一次完成。
  • 📝 閱讀理解題:文末 2–3 題(附中文解析),答完記錄最佳成績。
  • 📌 今日閱讀:複習完當天的字之後,自動推薦「用到最多今天複習單字」的文章,趁記憶還熱馬上再遇到它們。
  • ➕ 加新文章:學過的新字累積夠多時,閱讀頁最下面可以一鍵產生指令,貼給 Claude 就會照規則幫你寫新文章(只用你學過的字)。
📖 文法

依目前模式顯示:日文看 N5 文法、多益看 Part 5/6、英文看日常文法。每個主題有白話講解 + 例句(可朗讀) + 練習題(附解析)。

  • 🔗 搭配專項①〜④(多益):Part 5 最愛考的「動詞+受詞」搭配題——四個選項中文意思都像,練「選出母語者習慣的那一個」(waive the fee、hold a meeting、meet the deadline…),每題附白話解析教你用「受詞屬性」判斷。
  • 答對率會被記錄。
  • 🔥 錯題加強:重練答錯過的題,答對就移出。
  • 🔁 文法複習:文法題也納入間隔重複,今天到期的題會列出來,答對把下次複習拉更長。
🇯🇵 五十音(日文模式)

日文模式導覽列會出現「五十音」:

  • 假名表:平假名/片假名,含清音、濁音、拗音,點任一格聽發音。
  • 練習:看假名選拼音、或看拼音選假名,可選清音/加濁音/全部。
💬 對話課 / 🎓 模考
  • 對話:整段生活對話,可整段或逐句朗讀、開關中譯。
  • 模考(多益):Part 1–7 完整 200 題(含照片描述題),聽力播題、閱讀限時、估算分數與詳解;可勾「🎧 擬真聽力模式」仿正式考自動連播。錯題可一鍵變單字卡。
📊 統計與設定

統計頁有:掌握度、學習日曆熱力圖、近 30 日複習量、證照目標進度、考前衝刺計畫,以及這些設定:

  • 複習演算法:目標保持率(85/90/95%)。
  • 參數深度優化:複習紀錄累積滿 1,000 筆後,會多出「🤖 產生優化指令」按鈕——按一下同時下載紀錄 CSV+複製指令,把指令貼到電腦的 Claude Code 送出,就會用你的真實記憶數據訓練專屬的演算法參數(建議在電腦瀏覽器操作,自動同步會把手機的紀錄合併過來)。
  • 🔊 發音設定:語速滑桿、語音選擇、多益語速說明。
  • 🔤 字級:小/標準/大/特大,全 App 套用。
  • 危險區:重置所有資料(需兩道確認)。
🛡️ 資料保護與備份(很重要)

你的學習資料只存在瀏覽器,請務必做好備份:

  • 匯出備份(牌組頁):存成檔案,是最保險的一份。太久沒匯出首頁會提醒。
  • 匯入(覆蓋):把目前資料完全換成備份檔內容,連目標證照、考試日期、發音等所有設定都照搬。適合換新手機或要還原時用——手機原本的資料會被取代,所以請確定要用這份備份。
  • 匯入(合併):把兩邊的單字與複習進度聯集,兩邊都不會弄丟(同一張卡以較新的複習狀態為準);但設定會保留手機自己的,不會套用備份裡的目標證照/考試日期/發音(因為像「選的語音」在不同手機不通用)。適合兩台裝置都在背、想把進度湊在一起時用。
  • 👉 一句話記:換新機或還原就用「覆蓋」;兩台裝置湊進度才用「合併」。
  • 資料保護(牌組頁):開啟持久化(Android 不會自動清資料)+ App 內每天自動備份,可一鍵還原救回誤刪/誤重置。
  • ☁️ GitHub 雲端備份(牌組頁):設定一次(私人倉庫+金鑰,教學在區塊裡)後,預設⚡ 自動同步——開 App 自動從雲端下載合併、學習停下來後自動上傳,兩台裝置幾乎不用管;也保留手動「上傳/下載並合併/下載並覆蓋」按鈕(要立刻換裝置或救資料時用)。金鑰只存各裝置本機,不會跟著備份流出。
  • 多使用者(右上👤):每個人資料完全獨立。
